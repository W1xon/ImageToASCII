## 💀 Обзор

**ImageToASCII** — консольный инструмент, который превращает изображения **и видео**
в ASCII-арт и сохраняет результат в удобных форматах.

Начиналось с картинок.  
В **v2.0** стало понятно, что этого мало —  
теперь страдают **кадры видео**, последовательно и без пощады.

---

## 🧠 Основные возможности

- 🖼️ Конвертация изображений в ASCII  
- 🎞️ Конвертация **видео в ASCII-видео**  
- Выбор палитры (Basic, Palette16, Palette7, Natural)  
- Настройка ширины ASCII-вывода  
- Сохранение:
  - изображений → PNG  
  - видео → ASCII-видео (через покадровую обработку)  
- Улучшенная производительность (v2.0)  

> Никакого UI.  
> Только консоль, символы и холодный расчёт.
